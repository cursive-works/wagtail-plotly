# Scatter plot

A scatter plot `Plot data` is structured with separate X and Y value pairs (X0, Y0), (X1,Y1) etc.

|          | X0          | Y0     | X1       | Y1     |...Xn     | ...Yn  |
|----------|:-----------:|:------:|:--------:|:------:|:--------:|:------:|
| **Name** | `Predicted` | -      | `Actual` | -      |          |        |
| **D0**   | 11          | 3      | 10       | 2.3    |          |        |
| **D1**   | 15          | 4.3    | 16       | 2      |          |        |
| **D2**   | 22          | 5      | 24       | 6      |          |        |
| **D3**   | 19          | 7      | 22       | 7.2    |          |        |
| **...**  |             |        |          |        |          |        |
| **Dn**   |             |        |          |        |          |        |
