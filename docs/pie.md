# Pie chart

This is the most simple table layout, and does not include headers. The `Name` column is the label for each pie slice, and `Data` the size of the slice.

| **Name** | **Data** |
|----------|:--------:|
| Jan      | 3        |
| Feb      | 4.3      |
| Mar      | 5        |
| Apr      | 7        |
| ...      |          |
